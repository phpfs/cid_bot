# cid_bot
Telegram @cid_bot to retrieve your ChatID!

## Usage
### Personal Chats
Send a message to @cid_bot or open *https://t.me/cid_bot*.    
### Groups
To retrieve a group's chatID, you have to add @cid_bot to your group and send any message.
